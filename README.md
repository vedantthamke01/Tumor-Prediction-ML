# Tumor Prediction — ML Web App

A logistic regression-based tumor classifier with a modern web UI.

## Features
- Predicts **Benign / Malignant** based on tumor size and density
- Interactive sliders + precise number inputs
- Live decision boundary plot
- Probability breakdown with animated bars

## Stack
- Python · scikit-learn · Flask · Matplotlib

## Run Locally
```bash
python3 app.py
```
Then open → http://localhost:5050
